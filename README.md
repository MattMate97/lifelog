# LifeLog 📋

A personal life tracking dashboard built with Python and Flask. LifeLog helps you manage and monitor your life in one place.

## Features

- 📋 **Tasks** — Add, complete and delete tasks
- 💰 **Finances** — Track income and expenses with running balance
- 🎯 **Goals** — Set goals with target dates and mark them complete
- 🎬 **Movies & Shows** — Watchlist with watched/unwatched tracking
- 📚 **Books** — Track reading status (to read, reading, finished)
- 🎮 **Games** — Track games by platform and play status
- 🎵 **Music** — Discover and track artists and bands
- 💪 **Fitness** — Log workouts with duration and total stats

## Tech Stack

- **Backend** — Python, Flask
- **Database** — SQLite
- **Frontend** — HTML, CSS, JavaScript
- **Version Control** — Git, GitHub

## Getting Started

1. Clone the repo
```
   git clone https://github.com/MattMate97/lifelog.git
```

2. Navigate into the project
```
   cd lifelog
```

3. Install dependencies
```
   pip install -r requirements.txt
```

4. Run the app
```
   python3 app.py
```

5. Open your browser and go to `http://127.0.0.1:5000`
